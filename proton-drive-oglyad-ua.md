## Як працює шифрування

**Google Drive:**
```
Файл → Завантаження → Google сервер (Google бачить файл) → Твій пристрій
```

**Proton Drive:**
```
Файл → Шифрування на твоєму пристрої → Proton сервер (зашифровано) → Твій пристрій
                                                ↑
                                    Proton НЕ може відкрити
```

**Що шифрується:**
- ✅ Вміст файлів
- ✅ Назви файлів і папок
- ✅ Мітки часу
- ✅ Метадані файлів

Навіть структура папок (які папки існують, як вони називаються) — зашифрована.
