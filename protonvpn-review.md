## Security Features

### Secure Core
ProtonVPN's signature feature. Routes traffic through privacy-friendly countries (Switzerland, Iceland, Sweden) before exiting. Even if exit server is compromised, attacker only sees traffic from Secure Core server — not your real IP.

**When to use Secure Core:** High-risk situations — journalists communicating with sources, activists, anyone whose location exposure could have serious consequences.

### Tor over VPN
Connect to Tor network through ProtonVPN without installing Tor Browser. Adds significant anonymity layer at the cost of speed. Useful for accessing .onion sites.

### NetShield (Ad/Malware Blocker)
DNS-level blocking of ads, trackers, and malware. Works across all apps, not just browser. Less comprehensive than NordVPN's Threat Protection but effective for basic protection.

### Always-On VPN + Kill Switch
Two separate features:
- **Kill switch** — blocks internet if VPN drops
- **Always-on VPN** — automatically reconnects VPN if it drops

For Ukrainian users dealing with power outages: enable both. When power returns and internet comes back, ProtonVPN reconnects automatically.

### Open-Source Clients
All ProtonVPN apps (iOS, Android, Windows, Mac, Linux) are open-source and available on GitHub. Independent security researchers regularly audit them. This level of transparency is rare in the VPN industry.
