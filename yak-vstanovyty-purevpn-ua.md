## Крок 4: Налаштування для України

### Увімкніть Kill Switch
**Windows:** Settings → Security → Kill Switch → увімкніть

Kill Switch блокує весь трафік якщо VPN розривається. Критично при нестабільному підключенні під час відключень електроенергії.

### Оберіть протокол WireGuard
**Windows/Android:** Settings → Protocol → WireGuard

WireGuard — найшвидший і найнадійніший протокол. Рекомендовано для більшості користувачів.

### Split Tunneling (опційно)
**Android:** Settings → Split Tunneling → оберіть додатки що йдуть через VPN

Корисно якщо хочете пускати тільки браузер через VPN, а банківські додатки — напряму.

### Якщо провайдер блокує VPN
Київстар, Vodafone або lifecell іноді блокують VPN-трафік:
1. Settings → Protocol → **Stealth** або **SSL**
2. Ці режими маскують VPN під звичайний HTTPS

Після підключення відкрийте [whatismyip.com](https://whatismyip.com) або [ipleak.net](https://ipleak.net):
- IP-адреса повинна показувати країну сервера (не Україну)
- DNS leak test — всі запити мають іти через VPN-сервер

## Часті проблеми та вирішення

**VPN не підключається**
→ Змініть протокол: Settings → Protocol → спробуйте OpenVPN або IKEv2
→ Перезапустіть додаток
→ Перевірте підключення до інтернету без VPN

**Повільна швидкість**
→ Оберіть сервер ближче: Польща або Румунія
→ Змініть протокол на WireGuard
→ Вимкніть і увімкніть VPN знову

**VPN відключається при відключенні електрики**
→ Kill Switch увімкнений — це правильна поведінка
→ Після повернення інтернету просто натисніть Connect знову
→ Або увімкніть Auto-Connect: Settings → Auto-Connect

**Не відкривається Netflix US**
→ Спробуйте інший американський сервер (Лос-Анджелес замість Нью-Йорк)
→ Очистіть кеш браузера
→ Увімкніть режим Incognito
