## Security & Privacy

**Encryption:** AES-256 for OpenVPN/IKEv2, ChaCha20 for Lightway — both industry standard.

**No-logs policy:** KPMG conducted independent audits in 2022 and 2023, confirming ExpressVPN does not store user activity logs. Audit reports are publicly available.

**TrustedServer technology:** ExpressVPN servers run entirely on RAM — no data is ever written to hard drives. When a server is rebooted, all data is wiped. This means even if a server is seized, there's nothing to recover.

**Jurisdiction:** British Virgin Islands — outside 5/9/14 Eyes surveillance alliances. No mandatory data retention laws.

**DNS leak protection:** Built-in. ExpressVPN runs its own DNS servers so DNS queries never leave the encrypted tunnel.
