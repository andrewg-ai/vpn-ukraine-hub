## Why Streaming Services Block VPNs

Netflix, BBC iPlayer, and Disney+ have licensing agreements that restrict content by country. They detect and block VPN IP addresses to enforce these restrictions.

How they detect VPNs:
- **IP reputation databases** — known VPN server IPs are blacklisted
- **Multiple users per IP** — normal VPN servers have thousands of users sharing one IP
- **DNS leak detection** — if DNS doesn't match VPN server location, it's flagged

How good VPNs bypass this:
- Regularly rotate server IP addresses
- Use residential IP pools that look like real users
- Dedicated streaming servers with IPs not yet blacklisted
