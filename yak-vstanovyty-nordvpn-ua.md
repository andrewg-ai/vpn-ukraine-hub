## Як встановити NordVPN на Windows

### Крок 1: Завантажити додаток
1. Перейдіть на `nordvpn.com/download`
2. Натисніть **"Download NordVPN"** для Windows
3. Збережіть файл `NordVPNInstaller.exe`

### Крок 2: Встановити
1. Запустіть `NordVPNInstaller.exe`
2. Натисніть **"Yes"** якщо Windows питає дозвіл
3. Оберіть мову → **"Install"**
4. Дочекайтесь завершення (1-2 хвилини)
5. Натисніть **"Finish"**

### Крок 3: Увійти в акаунт
1. Відкрийте NordVPN
2. Натисніть **"Log In"**
3. Введіть email і пароль від NordVPN акаунту
4. Або увійдіть через браузер — натисніть **"Continue in browser"**

### Крок 4: Підключитися
1. Натисніть велику кнопку **"Quick Connect"**
2. NordVPN автоматично обере найшвидший сервер
3. Статус зміниться на **"Connected"** — зелений індикатор

**Готово!** Весь трафік тепер зашифрований.

## Як встановити NordVPN на Android

### Крок 1: Google Play
1. Відкрийте **Google Play Store**
2. Знайдіть **"NordVPN"**
3. Переконайтесь що розробник — **"nordvpn s.a."** (офіційний)

1. Відкрийте NordVPN
2. Натисніть **"Log In"**
3. Введіть email і пароль

## Які налаштування обрати після встановлення

### 1. Kill Switch — ОБОВ'ЯЗКОВО увімкнути
**Налаштування → Kill Switch → Увімкнути**

Kill Switch блокує інтернет якщо VPN раптово відключиться. Захищає від випадкового витоку реального IP.

### 2. Автозапуск при старті системи
**Налаштування → Загальні → Запускати при старті системи → Увімкнути**

VPN буде активний з першої секунди після увімкнення комп'ютера.

### 3. Auto-connect при підключенні до WiFi
**Налаштування → Auto-connect → Увімкнути**

Автоматично підключається до VPN при підключенні до будь-якої WiFi мережі. Критично для публічних точок доступу.

### 4. Протокол — NordLynx
**Налаштування → Протокол → NordLynx**

NordLynx (на основі WireGuard) — найшвидший протокол NordVPN. Рекомендований для більшості користувачів.

### 5. Obfuscated Servers — якщо провайдер блокує VPN
**Якщо стандартне підключення не працює:**
- Змініть протокол на **OpenVPN (TCP)**
- Увімкніть **Obfuscated Servers** в налаштуваннях
- Сервери будуть маскувати VPN трафік під HTTPS

## Вирішення проблем

### VPN не підключається
1. Перевірте інтернет без VPN — чи є підключення?
2. Спробуйте інший протокол: **Налаштування → Протокол → OpenVPN TCP**
3. Увімкніть **Obfuscated Servers**
4. Спробуйте інший сервер (інша країна)
5. Перезапустіть додаток

### Повільна швидкість
1. Змініть сервер на найближчий (Польща, Румунія)
2. Змініть протокол на **NordLynx**
3. Перевірте чи не завантажений вибраний сервер — спробуйте інший в тій самій країні

### VPN відключається само по собі
1. Увімкніть **Kill Switch**
2. На Android: увімкніть **Always-on VPN** в системних налаштуваннях
3. Перевірте налаштування економії батареї — деякі телефони вбивають VPN у фоні

### Сайти думають що я в іншій країні
Це нормально і правильно — VPN замінює ваш IP адрес. Якщо потрібен доступ до українських сервісів — увімкніть **Split Tunneling** і додайте ці сайти в виключення.

**Як встановити NordVPN безкоштовно?**

**Чи безпечно встановлювати NordVPN?**
Так. NordVPN — перевірений сервіс з аудитом від PricewaterhouseCoopers. Завантажуйте тільки з офіційного сайту nordvpn.com або офіційних магазинів (Play Store, App Store).

**Як перевірити що VPN працює після встановлення?**
Зайдіть на `whatismyip.com` — він покаже IP адресу. Якщо IP відрізняється від вашого реального українського IP — VPN працює.

**Як вимкнути NordVPN?**
Натисніть кнопку підключення ще раз — вона розкидається з "Connected" на "Disconnected". Або вийдіть з програми.

**NordVPN гальмує інтернет?**
На польському та румунському серверах втрата швидкості мінімальна — 2-5%. На далеких серверах (США, Азія) — 20-40%. Для стрімінгу і роботи в Європі гальмування практично не помітне.
